# Antibiotic Resistance Detection System

این پروژه یک مدل یادگیری ماشین (Random Forest) برای پیش‌بینی مقاومت یا حساسیت باکتری‌ها نسبت به آنتی‌بیوتیک است.

## 🚀 ویژگی‌ها
- پیش‌پردازش داده‌های آزمایشگاهی.
- استفاده از الگوریتم Random Forest Classifier.
- رسم نمودارهای تحلیلی (Confusion Matrix, Distribution, etc).
